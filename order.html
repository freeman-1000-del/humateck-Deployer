유튜브 AI 챗번역 FAQ

1. 문) 동영상 기본어 설정을 분명히 자국어로 했습니다. 그런데 다국어 번역컨텐츠 등록 후 확인해보니, 동영상 언어 설정이 미국 영어로 변했습니다. 이유가 뭘까요?

답) 다국어 번역컨텐츠 등록시 유튜브 다국어 번역컨텐츠 표시시스템상 동영상 기본설정이 미국 영어로 고정되어 있기 때문입니다. 그러나 이는 오류가 아니며, 고객님의 자국어 제목이나 설명문은 언제든지 자국어로 수정하실 수 있사오니 착오 없으시기 바랍니다.

7. 문) 이 등록시스템이 혹시나 등록 오류가 잦을까봐 선뜻 쓰기가 꺼려집니다.

답) 고객님의 그 마음 충분히 이해합니다. 그러나 시스템이 너무 심플해서, 어떤 불가항력적인 외부 악조건이 없는 한 저희의 어떤 훼방도 없이 그 어떤 컨텐츠도 곧바로 유튜브에 등록되는 시스템이오니 걱정하지 않으셔도 될 것 같습니다.

중요설정 길잡이

본 인증은 방문객들에게 AI 챗번역과 유튜브 등록의 실제를 체험하도록 허용합니다. 그러나 실제 등록은 구독자에게만 허용되오니 착오 없으시기 바랍니다.

처음 사용자를 위한 Google OAuth 승인방법입니다.

처음에는 조금 낯설 수 있습니다. 하지만 이 설정은 고객님의 유튜브 채널에 다국어 번역콘텐츠를 등록하기 위해 필요한 Google 공식 승인 절차입니다.

Google OAuth Client ID는 설정 후 상당기간 재사용되는 것이 일반적이지만, 사용량 과다로 인한 토큰 만료, 기타 이유로 재설정이 필요할 수도 있습니다.

1단계. Google Cloud로 이동해 주세요

아래 버튼을 눌러 Google Cloud 설정 화면으로 이동해 주세요.

Google Cloud 바로가기

Google Cloud에 접속한 뒤, 유튜브 등록에 사용할 고객님의 Google 계정으로 로그인해 주세요.

2단계. YouTube Data API v3 사용 설정

Google Cloud에서 YouTube Data API v3를 검색한 뒤 사용 설정을 해 주세요.

이 API가 켜져 있어야 유튜브 영상의 제목·설명문 다국어 등록 요청을 보낼 수 있습니다.

3단계. Branding 정보를 입력해 주세요

왼쪽 메뉴에서 Branding을 눌러 주세요.

  • App name에는 고객님이 알아보기 쉬운 이름을 적어 주세요.
  • User support email에는 고객님의 Google 이메일을 선택해 주세요.
  • Developer contact information에도 연락 가능한 이메일을 적어 주세요.

4단계. Target / Test users를 확인해 주세요

왼쪽 메뉴에서 Target을 눌러 주세요.

테스트 상태로 사용하는 경우에는 Test users에 실제 인증을 진행할 Google 이메일을 넣어 주세요.

5단계. OAuth Client를 Web application으로 만들어 주세요

왼쪽 메뉴에서 Client를 누른 뒤 Create client를 선택해 주세요.

Application type은 반드시 Web application으로 선택해 주세요.

6단계. Authorized JavaScript origins를 등록해 주세요

홈페이지에서 직접 Google 인증을 진행하려면 현재 사용하는 홈페이지 주소를 등록해야 합니다.

  • 정식 사이트 예: https://humateck.com
  • www 주소를 쓰는 경우: https://www.humateck.com
  • 로컬 테스트 시: http://localhost:5500 또는 실제 사용 중인 localhost 주소

끝에 / 를 붙이거나, /order.html 같은 경로를 넣으면 안 됩니다.

7단계. OAuth Client ID를 복사해 주세요

생성된 Client ID를 복사한 뒤, 주문페이지의 Google OAuth Web Client ID 입력칸에 붙여넣어 주세요.

입력칸은 보안을 위해 마스킹되어 보일 수 있습니다.

8단계. Google 인증하기를 눌러 승인해 주세요

주문페이지에서 Google 인증하기 버튼을 누르면 Google 공식 인증창이 열립니다.

해당 Google 계정이 등록하려는 유튜브 채널 또는 영상에 수정 권한을 가지고 있어야 정상 등록이 가능합니다.

중요 안내

여러 채널을 운영하더라도 같은 Google 계정이 해당 채널들의 관리 권한을 가지고 있다면, 각 영상의 실제 유튜브 영상 주소를 입력해 등록을 진행할 수 있습니다.

Google OAuth Client ID는 상당기간 재사용되는 것이 일반적이지만, 사용량 과다로 인한 토큰 만료나 기타 이유로 재설정 또는 재승인이 필요할 수 있습니다.

이는 서비스 오류가 아니라 Google 계정 보호와 YouTube API 사용을 위한 정상 보안 절차입니다.
